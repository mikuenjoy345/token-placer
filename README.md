# Miku's Token Placer
Ever wanted a self hosted token placing app/website?  Well, Miku's got ya covererd!  Mostly.

## Features
* Ad-Hoc token placement, via the 'Share Room' button.

## Requirements
* Perl
* Mojolicious
* JSON

Install these somehow.

### Debian/Ubuntu
```
# apt install libmojolicious-perl libjson-perl
```

### Other distros
Idk lol

## Start
```
$ cp ./token-placer.sh.example ./token-placer.sh
$ vim ./token-placer.sh                            # modify to your needs
$ ./token-placer.sh
```
